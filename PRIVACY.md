# Privacy Policy

This Privacy Policy explains how this Discord bot handles data.

## Data Collected
The bot may collect and store:
- Discord User IDs
- Discord Server IDs
- Command usage data

This data is collected only to provide bot functionality.

## Data Usage
Collected data is used to:
- Execute commands
- Maintain settings
- Improve bot performance

## Data Sharing
The bot does not sell, trade, or share data with third parties.

## Data Retention
Data is stored only as long as necessary for functionality.

## Data Removal
Users may request data deletion by contacting the bot owner.

## Changes
This policy may be updated at any time.
